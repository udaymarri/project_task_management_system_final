# TaskFlow - Project Management System

TaskFlow is a comprehensive project management dashboard designed to streamline collaboration and task tracking for teams. Built with a modern tech stack, it provides a seamless experience for administrators, managers, and team members.

## Features

- **Dynamic Dashboards**: Tailored views for different user roles.
- **Project Tracking**: Manage multiple projects with progress visualization.
- **Task Management**: Kanban-style task tracking with status monitoring.
- **Team Collaboration**: Invite members and manage team roles.
- **Notifications**: Real-time updates on task deadlines and project activities.
- **Dark Mode**: Fully responsive design with theme support.

## Tech Stack

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS
- **State Management**: Zustand (with Persistence)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **UI Components**: Shadcn/UI & Radix UI

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or pnpm

### Installation

```bash
# Clone the repository
# git clone <repository-url>

# Install dependencies
npm install
```

### Development

```bash
# Start the development server
npm run dev
```

### Build

```bash
# Build for production
npm run build
```
